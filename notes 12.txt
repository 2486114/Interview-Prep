JavaScript provides a comprehensive set of methods for interacting with and manipulating the Document Object Model (DOM), allowing dynamic changes to the structure, content, and style of web pages.
1. Element Selection Methods:
	document.getElementById('id'): Selects a single element by its unique ID.
	document.getElementsByClassName('className'): Returns an HTMLCollection of elements with a specific class name.
	document.getElementsByTagName('tagName'): Returns an HTMLCollection of elements with a specific tag name.
	document.querySelector('selector'): Returns the first element that matches a CSS selector.
	document.querySelectorAll('selector'): Returns a NodeList of all elements that match a CSS selector.
2. Element Creation and Insertion Methods:
	document.createElement('tagName'): Creates a new HTML element.
	parentNode.appendChild(childNode): Adds a child node to the end of a parent node's list of children.
	parentNode.insertBefore(newNode, referenceNode): Inserts a new node before a specified reference node as a child of a parent node.
	element.insertAdjacentHTML('position', htmlString): Inserts HTML content at a specified position relative to an element.
3. Element Removal Methods:
	parentNode.removeChild(childNode): Removes a specified child node from the DOM.
	element.remove(): Removes the element itself from the DOM.
4. Content and Attribute Manipulation Methods:
	element.innerHTML: Gets or sets the HTML content within an element.
	element.textContent: Gets or sets the text content of an element, stripping out any HTML tags.
	element.setAttribute('attributeName', 'value'): Sets the value of an attribute on an element.
	element.getAttribute('attributeName'): Gets the value of a specified attribute.
	element.removeAttribute('attributeName'): Removes a specified attribute from an element.
5. Style Manipulation Methods:
	element.style.propertyName: Directly accesses and modifies individual CSS properties of an element.
	element.classList.add('className'): Adds a class to an element's class list.
	element.classList.remove('className'): Removes a class from an element's class list.
	element.classList.toggle('className'): Toggles a class on and off.
	element.classList.contains('className'): Checks if an element has a specific class.
6. Event Handling Methods:
	element.addEventListener('eventType', handlerFunction): Attaches an event listener to an element.
	element.removeEventListener('eventType', handlerFunction): Removes an event listener from an element.
